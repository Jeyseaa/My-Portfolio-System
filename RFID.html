Bicutan Parochial School Attendance System using RFID with SMS

Project information

  • Category: Low Frequency (LF), High Frequency (HF) and Ultra-High Frequency (UHF).
  • Client: Bicutan Parochial School
  • Project date: March, 2019
  • Project URL: Bicutan Parochial School Attendance System(RFID)

This project is to create a system that can automatically capture students attendance by flashing their student NFC card at the RFID reader. SMS will then be sent to parents, which might help the school track students' attendance in and out and keep parents updated if their children are attending their class in school. It will also be applied to some private schools that don’t allow students to go out if it is not dismissal time. So the RFID Attendance Machine will be placed on the guard house.
